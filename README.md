# 📊 AI Impact on Workforce – Power BI Dashboard

Este projeto apresenta um **dashboard interativo desenvolvido no Power BI** para analisar os impactos do uso de **Inteligência Artificial no ambiente de trabalho**, com foco em métricas como **burnout, satisfação profissional, salários e adoção de ferramentas de IA**.

O objetivo do projeto é transformar dados em **insights visuais claros** que ajudem a entender como a adoção de IA está afetando profissionais de tecnologia em diferentes países e indústrias.

---
## 🔍 Principais Insights

Alguns insights identificados no projeto:

- A média geral é de aproximadamente **4 ferramentas de IA utilizadas por dia**
- A indústria **SaaS** apresentou o **maior uso de IA e também maior nível médio de burnout**
- O **Brasil apresentou o maior uso médio de IA por dia**
- Os salários variam significativamente entre **países, indústrias e níveis educacionais**
- A média geral de **satisfação no trabalho foi de 33,35**
